    수학

    포함-배제 등 중고등 수학 응용

    1. 진수와 진법 : 2진수, 10진수, N진수

      숫자로 구성된 문자열을 N진법에 맞게 변환하기?

      이론적인 방법 ; 각자리수 * N의 거듭제곱 합 - 거듭제곱 때문에 연산량이 많아짐

       def stoi (s, n) :
 	ret = 0
 	l = len(s)
 	for i in range (l) : ret += int(s[i]) * n ** (l-i-1)
 	return ret

      보다 간편한 방법

       def stoi (s, n) :
 	ret = 0
 	for i in s : ret = ret*n + int (i)
 	return ret

    2. 거듭 제곱 연산

      거듭제곱을 하게 되면 시간복잡도 급증 - 시간초과

      그래서 이진수를 활용! - 거듭제곱은 거듭제곱 간의 곱으로 표현 가능

    3. 최대공약수 GCD & 최소공배수 LCM

      최대공약수 GCD를 구하면 최소공배수 LCM은 0(1)에 구할 수 있다

      GCD 구하는 2가지 방법 - 둘 다 정식 풀이법은 아님

       # 1부터 체크
 def gcd (a, b) :
 	ret = 0
 	for i in range (min (a, b)) :
 		if a % i == and b % i == 0: ret = i
 	return ret

      min (a, b) 부터 체크 - 시간복잡도가 min(a, b) 로 정해져 있어 조금더 빠름

       def gcd (a, b) :
 	for i in range( min (a, b), 0, -1 ) :
 		if a % i == 0 and b % i == 0 :
 			return i

      유클리드 호제법 - 그냥 외워서 사용하는 것을 추천

       def gcd (a, b) :
 	return b if a % b==0 else gcd(b, a % b)

    4. 소수와 소인수 분해

    여러가지 소수 판별법과 소인수 분해

    2부터 N-1까지 ; O(N)

    def isPrime (N) :
	for i in range (2, N) :
		if N % i == 0 : return False
	return True

    2부터 sqrt(N)까지 ; O(sqrt(N))

    def isPrime (N) :
	i = 2
	while i * i <= N :
		if N % i == 0 : return False
		i += 1
	return True

    에라토스테네스의 체

    1 제거 - 2를 제외한 2의 배수 제거 - 3을 제외한 3의 배수 제거 - 5를 제외한 5의 배수 제거..

    몇 가지만 제거해도 상당수 숫자가 제거됨

    N까지 소수를 구하기 위해서 sqrt(N)까지 소수를 이용하면 가능!

    def era (N) :
	ck, p = [False for _ in range(N+1)], []
	for i in range(2, N+1) :
		if ck[i] == True : continue
		p.append(i)
		for j in range(i*i, N+1, i) :
			ck[j] = True
	return ck, p

    분할 정복 Divide & Conquer

    문제 분해, 재귀함수 활용

    1. 하노이탑

      간단한 정리

      1. 마지막 기둥이 움직이기 위해서는 나머지 모든 것이 한 곳에 있어야
      2. 그러면 N개 중 N-1개를 기둥 2에 보내는 것이 우선
      3. N번째 기둥을 3번에 보내고
      4. N-1개 기둥을 다시 기둥 3에 보낸다
      5. 기둥 1에서 기둥2로 보내는 과정과 기둥2에서 기둥3으로 보내는 과정은 같아
       def hanoi (st, ed, sz) :
 	if sz == 1 : return print(st, ed)
 	hanoi (st, 6-st-ed, sz-1)
 	print (st, ed)
 	hanoi (6-st-ed, ed, sz-1)

 n = int (input())
 print(2**n-1)
 hanoi(1, 3, n)
