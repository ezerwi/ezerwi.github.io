    Cookie & Session

    Cookie : client의 web browser에 임시저장 Session : server에 임시저장

    web project에서 session 특히 중요


    Java Web에서의 cookie & session

    Cookie

    Cookie coo = new Cookie ("cookie_name", "cookie_value"); 

cookie.setMaxAge(3600) // 1시간 수명, parameter는 초 
cookie.setMaxAge(0) // 바로 삭제 
cookie.setMaxAge(-1) // browser 종료와 함께 삭제, cookie의 기본 수명

response.addCookie(coo);

Cookie cookies[] = request.getCookies();

String name = coo.getName(); // cookie의 이름 
String value = coo.getValue(); // cookie의 값

    자동 로그인, 팝업창 ‘더 이상 보지 않기’ 등

    Session

    import javax.servlet.http.HttpSession;

HttpSession session = request.getSession(); 
// javax.servlet.http.HttpSession type의 객체 
// 새로운 session 시작 또는, 기존 session 정보를 객체로 만들어 return 

session.setAttribute("session_name", "session_value"); 
// session data 영역에 data 저장 

String str = (String) session.getAttribute("session_name"); 
// 저장된 data 호출 
// 기본적으로 Object type으로 저장되어 있으므로, 형변환casting 필요 

session.removeAttribute("session_name"); 
// 삭제 가능 : cookie와 차이점 

session.invalidate(); 
// session 종료
    • jsp에는 session이 내장되어있으나,
    • servlet에서는 getSession()을 통해 객체 생성 필요


    Flask에서의 cookie & session

    Cookie

    @app.route('/setcookie', methods=['POST','GET'])
def setcookie():
    if request.method == 'POST': 
        user = request.form['nm']
    resp = make_response("Cookie Setting Complete")
    resp.set_cookie('userID', user)
    return resp

@app.route('/getcookie')
def getcookie():
    name = request.cookies.get('userID')
    return f"<h1>welcome {name}!</h1>"

    Session

    • flask에서 session은 dictionary type으로 저장
    • key를 통해 호출
    • 해당 값을 암호화 하기 위한 key 값을 코드에서 지정해줘야 함
    from flask import session, redirect, url_for

app.secret_key = 'sesesesesisisisionononon'

@app.route('/')
def index(): 
    if 'username' in session :
        username = session['username']
        return f"<h1>Logged in as {username}</h1><br><br><a href = '/logout'>click to LOGOUT</a>"
    return f'<h1>You aren`t logged in</h1><br><br><a href ="/login"> click to LOGIN </a>'

@app.route('/login', methods = ['GET','POST'])
def login():
    if request.method == 'POST':
        session['username'] = request.form['username']
        return redirect(url_for('index'))
    return '''
        <form action = "" method = "post">
            <p><input type = "text" name = "username" /></p>
            <p><input type = "submit" value = "Login"/></p>
        </form>    
        '''

@app.route('/logout')
def logout():
    session.pop('username', None)
    return redirect(url_for('index'))

    session 유지 시간 정하기

    • session.permanent : default = False, 30일간 유지
    from datetime import timedelta

@app.before_request
def make_session_permanent():
    session.permanent = True
    app.permanent_session_lifetime = timedelta(minutes=1)


    Redirection & Error

    redirect()

    • 특정 url로 강제 이동
    • redirect(location, code=302, Response=None)

    abort()

    • HTTP Exception
      • 401 : Unauthorized
      • 404 : Not Found
      • 500 : Internal Server Error; Syntax
    • abort(status: Any, *args: Any, **kwargs: Any) -> NoReturn
    from flask import abort

@app.route('/login', methods = ['GET','POST'])
def login():
    if request.method == 'POST' :
        if request.form['username'] == 'admin':
            adminname = request.form['username']
            return redirect(url_for('success'))
        else : abort(401)
    return redirect(url_for('index'))

    @app.errorhandler(404)

    404 page 직접 만들기

    @app.errorhandler(404)
def page_not_found(error):
    return render_template('page_not_found.html'), 404
