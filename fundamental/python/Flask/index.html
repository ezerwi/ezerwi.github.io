    2. Flask App

    routes.py

    Basic

    from flask import Flask

app = Flask(__name__)

@app.route('/')
def hello_world():  return 'Hello World!'

if __name__ == '__main__' : app.run()

    app = Flask(__name__)

    Flask 클래스 전역 객체 선언

    @app.route('/')

    decorator(Java의 annotation) 사용하여 hello_world()를 wrapping

    웹으로 접근한 사용자가 어떤 URL에 따라 어떤 함수를 실행켜야할지 지정

    '/' : URL pattern, base url

    app.run()

    run()를 사용해 Flask app. 실행

    아무런 인자 넘겨주지 않을 경우, localhost (127.0.0.1)에서 적당한 port(여기서는 5000) 잡아서 실행

    Args

    • port : ex. (port = '5000')
    • host : ex. (host = '0.0.0.0')
      • 0.0.0.0은 모든 ip 의미하므로, 외부에서도 접속 가능
    • debug :

    username page

    @app.route('/user/<username>')
def show_user_profile(username):  
    return f'USER = {username}'

    post page

    @app.route('/post/<int:post_id>')
def show_post(post_id):  
    return f'Post ID = {post_id}'
    • Parameter의 data type 지정
    • 여기서 integer로 지정했기 때문에, 다른 type이 들어오면 “404 Not Found” page
    • parameter 변수 규칙에는 string, int, float 등이 있으며, default는 string
    • 특정함수 1개에 복수의 decorator 가능
      • ex. parameter 없으면 첫번째 url, 있으면 두번째 url
        @app.route('/hello/')
  @app.route('/hello/<name>')
  def hello(name=None):
      return f"User {name}"

    template

    Flask에서 html 사용하는 방법

    templates directory

    html 따로 보관하는 directory 만들어줘야 해당 파일들 인식 가능

    Jinja

    • template engine
    • html 문서 내 직접 코드를 실행시키는 문법
    • jsp의 <% %>와 유사
    types
    • {% CODE %} : delimeter for statements like if or for
    • {{ CODE }} : delimeter for expressions or variables
    • {# CODE #} : comments
    • {%- CODE %}, {%+ CODE %}, {% CODE -%} … :
      • 공백 제거 or 유지 (trim 기능)
      • 기본적으로 jinja engine은 template 줄 끝의 &nbsp;를 제외한 나머지 공백 그대로 출력, 이러한 공백을 제외하고 싶을 경우 - 붙이고 유지하고 싶으면 + 붙임
    • {% raw %} ...text... { % endraw %}
      • escape, template 형태와 동일한 텍스트 출력시 사용
      • 작은 문자열인 경우 {{ 와 같이 따옴표로 간단하게 escape 하는 것이 편리
    • # ... ## : line
    for in loop in Jinja
    
{% for (item) in (iterable object) %}
  CODES
{% endfor %}

 <!-- ex -->
{% for item in navigation %}
  <li><a href = "{{ item.href }}"> {{item.caption}} </a> </li>
{% endfor %}
    if statements in Jinja
    
{% if CONDITION %}
  CODES
{% elif CONDITION %}
  CODES
{% else %}
  CODES
{% endif %}

{% TRUE_CODES if CONDITION else FALSE_CODES %}
    for in if in Jinja
    
{% for ITEM in ITERABLE_OBJECT if CONDITION %}
  {{ ITEM.VALUE }}
{% endfor %}

    URL 및 HTTP

    redirect() & url_for()

    • 호출할 함수와 params 전달
    • url_for()에서 함수명만 전달할 경우 redirect()로 url 전달하는 것과 동일
    @app.route('/user/<name>')
def hello_user(name):
    if name == 'admin': 
      # return redirect(url_for('hello_admin'))과 동일
      return redirect('/admin')
    else : return redirect(url_for('hello_guest', guest = name))

    Get & Post

    @app.route('/letsin/')
def letsin ():

@app.route('/login', methods = ['POST', 'GET'])
def login():
    if request.method == 'POST': 
        user = request.form['nm']
        return redirect(url_for('success', name = user))
    else:
        user = request.args.get('nm')
        return redirect(url_for('success', name = user))
